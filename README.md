## 🚀 Projetos em Destaque

<br>

### 🥗 Sistema de Avaliação Nutricional - Estágio
> **Stack:** Flask · SQLAlchemy · SQLite · JavaScript · Jinja2 · Tailwind CSS

Sistema web completo para avaliação e acompanhamento de pacientes com **dashboard interativo** e gerenciamento de prontuários. Reduziu em **90%** o tempo de gestão de avaliações nutricionais em ambiente clínico.

**Funcionalidades:**
- Cadastro e histórico completo de pacientes
- Cálculo automatizado de indicadores antropométricos
- Dashboard com visualização de dados e evolução
- Arquitetura MVC com ORM e banco relacional

---

### 🏷️ Automação de Etiquetagem de Produtos - Estágio
> **Stack:** Python · FPDF · Customtkinter · datetime

Sistema desktop para categorização e impressão automática de etiquetas para produtos **Perecíveis, Semi-Perecíveis e Não-Perecíveis**. Reduziu em **95%** o tempo de elaboração de etiquetas em ambiente de produção.

---

### 📄 Automação de Documentos (Word & PDF) - Pessoal
> **Stack:** Python · python-docx · PyPDF2 / pdfplumber

Scripts e ferramentas para **processamento em lote** de documentos: formatação, extração de texto/tabelas/imagens, união, separação e conversão entre formatos. Destinado ao uso pessoal para organização de materiais acadêmicos (slides, trabalhos, livros) e estudos individuais.

---

### 📊 Análises Epidemiológicas e Nutricionais - Estágio
> **Stack:** Python · NumPy · Pandas

Projetos de análise de dados durante estágios em **hospital** e **ILPI**, incluindo:
- Avaliação antropométrica (IMC e Circunferência Panturrilha)
- Diagnóstico de risco nutricional, baixo peso e sarcopenia
- Medidas de tendência central, dispersão e prevalência
- Classificação do estado nutricional de populações
